Lab Description

In this lab, we worked on a real loan dataset to build classification models that predict whether a borrower will fully repay a loan or not.
We started by exploring the dataset using statistical summaries and visualizations such as histograms, count plots, and joint plots to understand the relationships between features like FICO score, interest rate, and loan purpose.

After that, we prepared the data by converting the categorical feature (purpose) into numerical dummy variables so it can be used in machine learning models. Then, we split the data into training and testing sets.

We trained two models: a Decision Tree and a Random Forest. Finally, we evaluated their performance using confusion matrix and classification report to compare accuracy and prediction quality.

⸻

What I Learned

* How to explore and understand real-world datasets using visualization tools
* How to preprocess data, especially handling categorical variables
* How to build and train Decision Tree and Random Forest models
* How to evaluate model performance using confusion matrix and classification report
* The advantage of Random Forest over Decision Tree in reducing overfitting and improving accuracy

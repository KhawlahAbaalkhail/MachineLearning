Customer Spending Prediction Using Linear Regression

This project focuses on building a regression model to estimate customer yearly spending based on their engagement data.



Data Overview

The dataset includes behavioral metrics such as:
	•	Average session duration
	•	Time spent on the mobile application
	•	Time spent on the website
	•	Membership duration

Output Variable
	•	Annual spending value per customer



Technologies Used
	•	Python
	•	Pandas & NumPy
	•	Matplotlib & Seaborn
	•	Scikit-learn



Methodology
	•	Load and examine the dataset structure
	•	Explore relationships between variables through visualization
	•	Prepare the input features and target variable
	•	Divide the dataset into training and testing subsets
	•	Fit a Linear Regression model
	•	Generate predictions
	•	Assess model performance using standard regression metrics
	•	Analyze results through visualizations


Key Observations
	•	Membership duration has the highest influence on spending behavior.
	•	User activity on the app contributes more to predictions than website usage.
	•	The model demonstrates reliable performance with minimal prediction error.

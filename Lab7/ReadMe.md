Customer Spending Prediction Using Linear Regression

This assignment focuses on applying Linear Regression to predict customer yearly spending based on their engagement data.

Data Overview

The dataset includes behavioral metrics such as:
• Average session duration
• Time spent on the mobile application
• Time spent on the website
• Membership duration

Output Variable

• Yearly amount spent by each customer

Technologies Used

• Python
• Pandas & NumPy
• Matplotlib & Seaborn
• Scikit-learn

Methodology

• Load and inspect the dataset
• Explore relationships between variables using visualizations
• Prepare features and target variable
• Split the data into training and testing sets
• Train a Linear Regression model
• Make predictions
• Evaluate model performance using regression metrics
• Visualize and interpret the results

Key Observations

• Membership duration shows the strongest impact on spending
• Time spent on the app is more influential than website usage
• The model provides good predictions with low error
